    mapStateToProps ES6
    Not a question, but a comment. We can take ES6 even further using fat arrow functions: const mapStateToProps = ({weather}) => ({weather}); export default connect(mapStateToProps)(WeatherList); Note: Parentheses are needed around the function result, otherwise it's ambiguous as to weather the braces represent a group of statements or an object. We can take the above even further: export default connect(({weather}) => ({weather}))(WeatherList);
